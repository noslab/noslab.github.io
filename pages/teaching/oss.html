CDA0156: Open Source Software

In this class, we deal with the basics of open-source software within a reasonable level of depth. The course divides fairly into two parts; the students can learn how to install and use Linux operating system, and the students will download the open-source code (including MySQL, Cassandra, Tensorflow, and Pytorch) and learn how to modify it.
