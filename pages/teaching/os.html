CDA0017: Operating Systems

This course is intended to make students be familiar with opeprating systems. In this class, we will end up understanding most of the important things an operating system does. The students will learn how the processes of a program execute in detail.

Text Books
  • Remzi Arpaci-Dusseau and Andrea Arpaci-Dusseau, Operating Systems: Three Easy Pieces, CreateSpace Independent Publishing Platform, 2018
References
  • Abraham Silberschatz, Greg Gagne, and Peter B. Galvin, Operating System Concepts, Wiley, 10th Edition, 2018
  • Robert Love, Linux Kernel Development, Addison-Wesley Professional, 3rd Edition, 2010
  • The Linux Kernel Archives [Website]
  • Linux Cross Rereference [Website]

Prerequisites

  • CDA0006: System Programming
